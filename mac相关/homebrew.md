# homebrew

### conform installed software
```
brew list
```

### install package
```
brew install package_name					  // CUI
brew cast install package_name			// GUI
```

### uninstall package
```
brew uninstall package_name
```

### update brew
```
brew update
```

### update installed packages
```
brew upgrade
```
### delete old package version
```
brew clearup
```

### search the package
```
brew search package_name
```
