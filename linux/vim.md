# VIM常用命令

###### find

命令模式下，输入 /搜索文字 回车。

查看下一个匹配按下n(小写n)

转到上一个匹配呢？按下N（大写N）。

```
/extension
```

![image-20210314214402002](https://i.loli.net/2021/03/14/PpNvD3TGW7OJ9gF.png)

###### 编辑模式

```
// insert
i
```

###### 退出模式

```
Esc
```

###### 保存&退出

```
// write  quit
:wq
```

###### 不保存&退出

```
// quit
:q!
```
